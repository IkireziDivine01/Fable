# Immersive prop models

Stylized low-poly `.glb` props used by the story stage.

Generated by `node scripts/generate-prop-models.mjs` (procedural geometry exported via Three.js `GLTFExporter`). License: same as the Fable project source (original assets, not third-party packs).

| File | Prop type |
|------|-----------|
| `hut.glb` | hut |
| `tree.glb` | tree |
| `banana_tree.glb` | banana_tree |
| `stall.glb` | stall |
| `rock.glb` | rock |
| `drum.glb` | drum |
| `water_jug.glb` | water_jug |
| `goat.glb` | goat |
| `bench.glb` | bench |
| `board.glb` | board |

FX props (`fire`, `flower`, `path`, `millet_field`) stay procedural in code.
